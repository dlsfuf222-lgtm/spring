# Json 데이터를 자바 클래스 객체로 변환하는 것은 언마샬링(ummarshalling) 입니다.

> 정리
> # 언마샬링(Unmarshalling)
* # JSON -> Java 객체
*  역직렬화(Deserialization)와 거의 동일한 의미
*  예 : Jackson # ObjectMapper.readValue(json, MyClass.class)
> # 마샬링(Marshalling)
* # Java객체 -> JSON
* 직렬화 (Serialization)와 동일한 개념
* 예 : Jackson # ObjectMapper.writhValueAsString(myObj)
> 요약 표

 작업 의미 방향 ## 마샬링(Marshalling) ## Java 객체를 전송/저장 가능한 포맷(JSON 등)으로 변환 Java->
JSON ## 언마샬링(Unmarshalling) ## JSON 등의 포맷을 Java 객체로 변환 JSON -> Java
